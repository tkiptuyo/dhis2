Under “System Variables” click “New” to add the following:
Variable Name: DHIS2_HOME
        Value: C:\DHIS2

Variable Name:  JAVA_HOME
        Value: C:\Program Files\Java\jre1.8.0_261

Variable Name: JAVA_OPTS
      Value: -Xms512m -Xmx1024m -XX:PermSize=256m -XX:MaxPermSize=768m

Make sure “Path” variable includes the path to JDK’s “bin” folders address (i.e. C:\Program Files\Java\jdk1.8.0_251\bin).
